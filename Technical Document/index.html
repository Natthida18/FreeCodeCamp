Introduction to HTML

HTML (Hypertext Markup Language) is a language used to create the structure of web pages on the World Wide Web. HTML is used to specify the structure of web pages and to format the content of websites. Therefore, HTML is essential for web development.

HTML documents are files that end with a .html or .htm extension. A web browser reads the HTML file and renders its content so that internet users can view it.

It’s also worth noting that HTML is not considered a programming language as it can’t create dynamic functionality, although it is now considered an official web standard.

The World Wide Web Consortium (W3C) maintains and develops HTML specifications.

What is HTML
  1. HTML stands for Hyper Text Markup Language
  2. HTML is the standard markup language for creating Web pages
  3. HTML describes the structure of a Web page
  4. HTML consists of a series of elements
  5. HTML elements tell the browser how to display the content

All HTML pages have a series of HTML elements, consisting of a set of tags and attributes. HTML elements are the building blocks of a web page. A tag tells the web browser where an element begins and ends, whereas an attribute describes the characteristics of an element.

Structure
  • <!DOCTYPE html>: Declaration of HTML5 document type.
  • <html>: The main tag of the HTML document.
  • <head>: The header part of the document used to specify document information such as character set used, title, or links to CSS files.
  • <meta>: Tags used to specify information about the document, such as the character set used and the screen size.
  • <title>: Tag used to set the title of the web page.
  • <body>: The main part of the web page content.
  • <header>, <nav>, <main>, <section>, <footer>: Tags used to specify different parts of the web page such as header, navigation, main content, sections, and footer.
  • <h1>, <h2>, <p>, <ul>, <li>: Tags used to specify headings, paragraphs, lists, sublists, and content of the web page.

The HTML code above serves as an example of the basic structure of a web page, which can be edited and customized according to the requirements of each website.

Images and Links
  • <img>: Used to display images on web pages.
  • <a>: Used to create links that connect different web pages.
Divisions
  • <div>: Used to create boxes or groups of content on web pages.
  • <span>: Used to group subtexts within <div> or <p>

Divisions help organize the content of a webpage into meaningful sections and provide structure for styling and layout purposes. They are fundamental building blocks in modern web development.

Embedding
  • <iframe>: Used to display content from external web pages within the current web page.
  • <audio> and <video>: Used to embed audio and video content in web pages respectively.
Conclusion

HTML is the primary markup language found on the internet. Every HTML page has a series of elements that create the content structure of a web page or application.

HTML is a beginner-friendly language with plenty of support and is mainly used for static website pages. HTML works best together with CSS for styling and JavaScript for functionality.

In HTML, divisions are represented by the <div> element. The <div> element is a block-level container that is commonly used to group together elements for styling purposes or to structure the layout of a web page. Divisions can be styled using CSS to control their appearance and layout on the page.
